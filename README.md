## Text Analyzer
Python skript pro autentizaci uživatelů a detailní statistickou analýzu textu.

## Funkce
- **Ověření uživatele:** Přihlašovací systém.
- **Analýza:** Počítá slova, velká/malá písmena a číselné hodnoty.
- **Graf:** Histogram délek slov přímo v terminálu.

## Spuštění
Bash
```
python analyzer.py
```
